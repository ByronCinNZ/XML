Metadata for Data Quality (MDQ) Version: 1.0

Description

MDQ 1.0 is an XML Schema implementation derived from ISO ISO 19157, Geographic Information - Data Quality, Clause 6.4. It includes elements for describing the quality of resources The XML schema was encoded using the rules described in ISO/TS 19139:2007, Clause 8 and ISO/TS19115-3, Clause 8.

XML Namespace for mdq 1.0

The namespace URI for mdq 1.0 is http://www.isotc211.org/namespace/mdq/1.0.

XML Schema for mdq 1.0

mdq.xsd is the XML Schema document to be referenced by XML documents containing XML elements in the mdq 1.0 namespace or by XML Schema documents importing the mdq 1.0 namespace. This XML schema includes (indirectly) all the implemented concepts of the mdq namespace, but it does not contain the declaration of any types.

NOTE: The XML Schema for mdq 1.0 are available here as part of a zip archive including all the XML Schema Implementations defined in ISO/TS 19115-3.

Related XML Schema for mdq 1.0

dataQuality.xsd implements the UML conceptual schema defined in ISO 19157, 6.4. It was created using the encoding rules defined in ISO 19118 and extended in ISO 19139 and contains the following classes: CoverageResult, DQ_AbsoluteExternalPositionalAccuracy, DQ_AccuracyOfATimeMeasurement, DQ_Aggregation, AbstractDQ_Completeness, DQ_CompletenessCommission, DQ_CompletenessOmission, DQ_ConceptualConsistency, DQ_Confidence, DQ_ConformanceResult, AbstractDQ_DataEvaluation, DQ_DataInspection, DQ_DataQuality, DQ_DescriptiveResult, DQ_DomainConsistency, AbstractDQ_Element, AbstractDQ_Evaluation, DQ_FormatConsistency, DQ_FullInspection, DQ_GriddedDataPositionalAccuracy, DQ_Homogeneity, DQ_IndirectEvaluation, AbstractDQ_LogicalConsistency, DQ_MeasureReference, DQ_Metaquality, DQ_NonQuantitativeAttributeAccuracy, AbstractDQ_PositionalAccuracy, DQ_QuantitativeAttributeAccuracy, DQ_QuantitativeResult, DQ_RelativeInternalPositionalAccuracy, DQ_Representativity, DQ_Result, DQ_SampleBasedInspection, DQ_Scope, DQ_StandaloneReportInformation, AbstractDQ_TemporalAccuracy, DQ_TemporalConsistency, DQ_TemporalValidity, AbstractDQ_ThematicAccuracy, DQ_ThematicClassificationCorrectness, DQ_TopologicalConsistency, DataQualityValueType, DataQualityValueUnit, QE_CoverageResult, AbstractQE_Usability, DQM_RegisteredDataQualityBasicMeasure, DQM_RegisteredDataQualityMeasure, and DQM_RegisteredDataQualityParameter
